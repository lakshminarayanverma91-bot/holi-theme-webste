# Holika Ki Holi 🌈

**Holika Ki Holi** is a Holi-themed interactive website that presents
the culture, traditions, and stories behind the festival of Holi through
an engaging digital experience.

The project combines **festival information, storytelling, interactive
features, and entertainment elements** to create a complete Holi-themed
web platform.

🔗 **Live Website:**\
https://holikakiholi.netlify.app/

------------------------------------------------------------------------

# Overview

This project was built as a creative web development experiment to
combine **culture, technology, and interactive design** in a single
platform. The website explains the significance of Holi while also
offering interactive sections such as games, quizzes, and personalized
wishes.

The goal of this project is to create a **festival-focused digital
experience** that is both informative and entertaining.

------------------------------------------------------------------------

# Features

## Holi Information

A detailed introduction to the festival of Holi, its cultural
background, traditions, and significance.

## Prahlad Story Timeline

A structured timeline explaining the story of Prahlad and the events
leading to Holika Dahan.

## Holika Dahan

Explanation of the ritual of Holika Dahan and its importance in the Holi
festival.

## Holi Dishes & Recipes

A section highlighting traditional Holi dishes along with their
preparation methods.

## Holi Wish Cards

Pre-designed Holi greeting cards that can be shared with friends and
family.

## AI Wish Generator

A feature that generates personalized Holi wishes.

## Holi Mini Games

Interactive games designed around the Holi theme.

## Holi Quizzes

Quiz section containing questions related to Holi traditions and
history.

## Holi Gallery

A gallery displaying images related to the Holi festival.

## Shop Section

A sample shop interface where users can explore Holi-related items.

> Currently this section contains frontend UI only.

## Holi Safety Tips

Important guidelines for celebrating Holi safely and responsibly.

## Meme Section

A dedicated section featuring Holi-related memes.

## Music Player

An integrated music player with multiple Holi songs and playback
controls.

## Multiple UI Modes

The website supports multiple visual themes including:

-   Light Mode
-   Dark Mode
-   Additional custom UI modes
